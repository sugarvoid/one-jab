# one-jab

Demake of One Punch. Using the Godot Engine. 


## How To Play

- When next to punching bag, punch it. 
- When away from punching bag, move to it.

## Controls :joystick: 

|Key|Action|
|:---:|:---:|
|Z|Move|
|X|Punch|
